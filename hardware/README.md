
# Hardware

## PCB Files
PCB Production files for the OpenDoorSim board can be found in the [PCB Files](/hardware/PCB Files/) folder.

> [!WARNING]
> When ordering the PCB, make sure the pin headers for the screen and encoder (placed so that through holes are generated for their wires) are not placed, as they are marked as DNP and should not be populated. Otherwise, you'll have a header where the wires need to go through!
> 

## 3D Models
3D STL models, as well as print instructions and settings for the Pocket and LAB can be found in the ![3D Models](/hardware/3D Models/) folder.

> [!WARNING]
> Print settings are for ASA. If you are not printing with ASA, then you will need to adjust your print settings!
>
