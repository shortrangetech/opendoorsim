# OpenDoorSim Firmware Guide

Welcome to the software folder for OpenDoorSim! The main firmware files live in this folder.

*For full instructions, please refer to the [Official Documentation](#).*

## Flashing Your OpenDoorSim's Firmware

OpenDoorSim's firmware can be flashed to your ESP32 device using the shell script (flash_board.sh) in the parent folder, or PlatformIO in your favorite IDE.
