# OpenDoorSim Software Guide

Welcome to the software documentation for OpenDoorSim! This guide covers the firmware, the Web UI, and how to flash your device.

## Firmware Features

- Beautiful Web UI, manage CTFs and card data with ease
- Hardware Menu (run it offline!)
- Multiple Modes for various use cases:
  - **Raw Mode**
  - **CTF Mode**
  - **Wifi Mode**
  - **Tamper Detect Mode** (for supported readers)
- Batch User Management

## Flashing Your OpenDoorSim

OpenDoorSim's firmware can be flashed to your ESP32 device using the provided scripts or PlatformIO.

*For full instructions, refer to the [Official Documentation on Mintlify](#).*

## Using the Web UI

The built-in Web UI makes it incredibly easy to configure your reader, view captured RFID codes, and manage users. Simply connect to the OpenDoorSim's WiFi access point once flashed, and navigate to the local IP address provided on the hardware screen.
